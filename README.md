# Supply-Chain-Inventory-Analytics-Dashboard
Interactive dashboards analyzing supply chain operations, inventory performance, and business insights.

This project focuses on analyzing supply chain operations and inventory performance to derive actionable business insights. Interactive dashboards were developed to monitor key metrics and improve operational efficiency.

Key Insights:
- Analyzed sales and order trends across different time periods
- Evaluated inventory distribution and product performance
- Assessed delivery performance and logistics efficiency
- Identified patterns in transportation modes and regional operations

Dashboard Features
Inventory Management Dashboard:
- Tracking of inventory performance and product distribution
- Category-wise and region-wise sales analysis
- Order and quantity trend analysis
  
Supply Chain Dashboard:
- Monitoring delivery performance and order status
- Analysis of transportation modes and logistics operations
- Regional and yearly order trend insights

Data Processing:
- Extracted and analyzed data using SQL queries
- Calculated key metrics related to sales, orders, and performance
- Aggregated and structured data for dashboard visualization

Tools & Technologies:
- Power BI
- Excel
- SQL
